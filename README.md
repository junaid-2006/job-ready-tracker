# 🚀 Job Ready Tracker

A **Notion-style roadmap tracker** for a 2nd-year CSE student — focused on:

- Core CS fundamentals  
- Cloud & DevOps  
- AI/ML + LLM apps  
- Full-stack development  
- Project-based learning  

---

## 📊 Overview

**Goal:** Become job-ready with structured learning + projects.

**System:**
- 🧠 5 days → Learning + Coding  
- 🛠️ 1 day → Project building  
- 🔁 1 day → Revision + Resume  

---

## 📌 Phases

---

### 🧩 Phase 1 · Programming + Linux (Months 1–3)

**Goal:** Strong coding + development workflow

- [ ] Pick one language (Python / Java)
- [ ] Master OOP concepts
- [ ] Use Linux daily (commands, processes, env, etc.)
- [ ] Learn Git & GitHub

**Mini Projects:**
- [ ] Student Management System (OOP-based CLI app)
- [ ] Linux System Monitor (CPU, memory, logs)

---

### 🗄️ Phase 2 · DBMS + SQL + NoSQL (Months 4–6)

**Goal:** Data design + querying skills

- [ ] DBMS fundamentals (ER, normalization, ACID)
- [ ] Practice 100+ SQL queries
- [ ] Learn MongoDB basics

**Mini Projects:**
- [ ] Library / Hostel Database
- [ ] SQL Analytics Backend
- [ ] MongoDB Notes App

---

### 🌐 Phase 3 · Networking + Web (Months 7–9)

**Goal:** Understand how systems communicate

- [ ] Networking basics (OSI, TCP/IP, HTTP, DNS)
- [ ] Frontend (HTML, CSS, JS)
- [ ] Build REST APIs (Flask / FastAPI / Java)

**Mini Projects:**
- [ ] URL / Network Analyzer
- [ ] Local Full-Stack Web App

---

### ☁️ Phase 4 · Cloud + DevOps (Months 10–12)

**Goal:** Deployment & scalability

- [ ] Cloud fundamentals
- [ ] AWS core services (EC2, S3, IAM, etc.)
- [ ] Docker & CI/CD basics

**Mini Projects:**
- [ ] Static site on S3
- [ ] Backend deployed on EC2
- [ ] CI/CD Pipeline

---

### 🤖 Phase 5 · AI/ML + LLM Apps (Months 13–18)

**Goal:** Build real AI applications

- [ ] ML basics (training, evaluation)
- [ ] Run local LLMs (Ollama / LM Studio)
- [ ] Learn RAG (retrieval + embeddings)
- [ ] Understand AI agents

**Mini Projects:**
- [ ] Local PDF Chatbot
- [ ] RAG Study Assistant

---
